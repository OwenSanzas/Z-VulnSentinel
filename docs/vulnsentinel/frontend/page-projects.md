## 列说明

| 列 | 说明 |
|------|------|
| **Project** | 项目名 + 仓库地址 |
| **Version** | 当前版本（tag 或 branch@commit） |
| **Deps** | 依赖库数量 |
| **Vulns** | 当前活跃 client_vulns 数（recorded + reported + confirmed） |
| **Since** | 开始监控日期 |
| **Last Update** | 最近一次代码更新时间 |

## 交互

| 操作 | 跳转 |
|------|------|
| 点击 `details →` | [项目详情页](page-project-detail.md) |
| 点击 Vulns 数字 | [漏洞列表页](page-vulnerabilities.md)，筛选该项目 |
