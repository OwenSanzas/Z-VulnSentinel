# Crawler

Per-platform crawlers for discovering and resolving library metadata (repo URL, versions, etc.) from package registries.

## Platforms

| Directory | Ecosystem | Registry |
|-----------|-----------|----------|
| `python/` | Python | PyPI |
| `java/` | Java / Kotlin | Maven Central |
| `go/` | Go | pkg.go.dev / proxy.golang.org |
| `rust/` | Rust | crates.io |
| `c_cpp/` | C / C++ | Conan Center, vcpkg, GitHub |
| `solidity/` | Solidity | Soldeer |
